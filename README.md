# VSCODE-course

Python 算法学习项目，记录在 VS Code 中学习编程的练习代码。

## 项目内容

### bubble_sort.py — 冒泡排序

实现经典冒泡排序算法，并附带示例运行。

**算法原理：** 重复遍历列表，比较相邻元素并交换，直到列表有序。

**时间复杂度：** O(n²)  
**空间复杂度：** O(1)

**运行示例：**

```bash
python bubble_sort.py
```

输出：

```
排序前: [64, 34, 25, 12, 22, 11, 90]
排序后: [11, 12, 22, 25, 34, 64, 90]
```

## 环境要求

- Python 3.x

```bash
git clone https://github.com/Ashley-wa/VSCODE-course.git
```
